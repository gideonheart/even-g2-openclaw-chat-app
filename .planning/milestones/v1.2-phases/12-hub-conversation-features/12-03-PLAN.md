      - Find the first case-insensitive match position: `text.toLowerCase().indexOf(query.toLowerCase())`

      - If query is empty or blank, return empty array
      - Lowercase the query once: `const lowerQuery = query.toLowerCase()`
      - First, load all conversations into a Map<id, name> via a readonly transaction on 'conversations' store (need the name for SearchResult)

      - For each message, check `msg.text.toLowerCase().includes(lowerQuery)`
      - If match, create a SearchResult with snippet from `extractSnippet` and the conversation name from the map
      - Stop collecting after reaching `limit` results
      - Return results sorted by timestamp descending (most recent first)
      - Note: This runs in a single readonly transaction over both 'conversations' and 'messages' stores
