**Step 3: Create `src/bridge/even-bridge.ts`**

Follow RESEARCH.md Pattern 1 exactly. Factory function `createEvenBridgeService(bus: EventBus<AppEventMap>): BridgeService`:

- `init()`: Call `waitForEvenAppBridge()` ã store bridge instance ã call `createStartUpPageContainer(STARTUP_LAYOUT)` (single container: containerID=1, name='main', 576x288, isEventCapture=1) ã register `onDeviceStatusChanged` (emit `bridge:connected` / `bridge:disconnected`) ã register `onEvenHubEvent` callback that:
  - Checks `event.audioEvent` ã emits `bridge:audio-frame` (add to AppEventMap: `'bridge:audio-frame': { pcm: Uint8Array; timestamp: number }`)
  - Reads `eventType` from `event.listEvent?.eventType ?? event.textEvent?.eventType ?? event.sysEvent?.eventType`
  - Maps: `CLICK_EVENT(0)` ã `gesture:tap`, `DOUBLE_CLICK_EVENT(3)` ã `gesture:double-tap`, `SCROLL_TOP_EVENT(1)` ã `gesture:scroll-up`, `SCROLL_BOTTOM_EVENT(2)` ã `gesture:scroll-down`
- `destroy()`: Call unsubscribe functions (null after call for idempotency) ã `shutDownPageContainer(0)` ã set bridge to null
- `startAudio()`: Call `bridge.audioControl(true)`, return result
- `stopAudio()`: Call `bridge.audioControl(false)`, return result

Add `'bridge:audio-frame': { pcm: Uint8Array; timestamp: number }` to `AppEventMap` in `src/types.ts`.

**Step 1: Create `src/audio/audio-capture.ts`**

Follow RESEARCH.md Pattern 4. Factory function `createAudioCapture(devMode: boolean): AudioCapture`:

Export interface:
```typescript
export interface AudioCapture {
  startRecording(sessionId: string): void;
  stopRecording(): Promise<Blob>;
  onFrame(pcm: Uint8Array): void;
  isRecording(): boolean;
}
```

Implementation:
- Internal state: `frames: Uint8Array[]`, `mediaRecorder: MediaRecorder | null`, `mediaChunks: Blob[]`, `recording: boolean`
- `startRecording(sessionId)`: Clear frames array, set recording=true. If devMode, call `navigator.mediaDevices.getUserMedia({ audio: true })`, create MediaRecorder with 100ms timeslice, collect chunks in mediaChunks.
- `onFrame(pcm)`: If recording and NOT devMode, push pcm to frames array. CRITICAL: This runs at ~100Hz (25 frames/sec). Must be fully synchronous ã no await, no async, just `frames.push(pcm)`.
- `stopRecording()`: Set recording=false. If devMode with mediaRecorder, return Promise that resolves with Blob({ type: 'audio/webm' }) on mediaRecorder.onstop. If glasses mode, concatenate all PCM frames into single Uint8Array ã return `new Blob([buffer], { type: 'audio/pcm' })`.
- `isRecording()`: Return recording flag.

**Step 2: Create `src/__tests__/audio-capture.test.ts`**

Test dev mode (devMode=true):
