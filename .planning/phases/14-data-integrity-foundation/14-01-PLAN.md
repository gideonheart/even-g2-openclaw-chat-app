   - Returns the new IDBDatabase

1. **getAllConversations():** Inside the cursor loop, add a guard:
```typescript
if (cursor) {
  const record = cursor.value as ConversationRecord;
  if (record.id !== SENTINEL_ID) {

  }
  cursor.continue();

```

2. **getLastConversation():** The cursor returns the most recently updated record. If it's the sentinel, skip it and continue to the next:
```typescript
cursorReq.onsuccess = () => {
  const cursor = cursorReq.result;
  if (cursor) {
    const record = cursor.value as ConversationRecord;
    if (record.id === SENTINEL_ID) {
      cursor.continue();
      return;
    }
    resolve(record);
  } else {
    resolve(undefined);
  }
};
```
